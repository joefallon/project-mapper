# Contributor Covenant Code of Conduct

We are committed to a welcoming and respectful community. All contributors are expected
to follow this code of conduct.

Be kind and respectful. Harassment, discrimination, or abusive language will not be tolerated.

If you experience or witness unacceptable behavior, please report it by opening an issue or
contacting the maintainers at joseph.t.fallon@gmail.com.

For the full text, consider adopting the Contributor Covenant (https://www.contributor-covenant.org/).
